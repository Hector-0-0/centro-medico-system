para iniciar la base de datos necesitan tener mysql 9.4.0 y luego iniciarlo 
configuren el archivo config.propierties en /desktop/src/main/resources/config.properties
debe tener esta forma: 
db.host=localhost
db.port=3306
db.name=centro_medico
db.user=root
db.password= 

solo cambia el password por el que tengas en tu base de datos
